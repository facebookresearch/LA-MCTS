Welcome to LA-MCTS's documentation!
===================================
**LA-MCTS** is a Python library for blackbox optimization. It's based of paper
`Learning Search Space Partition for Black-box Optimization using Monte Carlo Tree Search
<https://arxiv.org/abs/2007.00708>`_.

.. note::
   This project is under active development.
